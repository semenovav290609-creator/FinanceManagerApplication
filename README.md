# FinanceManagerAppelication 💰

Простое, аккуратное и быстрое десктопное приложение для управления личными финансами. Проект представляет собой успешный перенос консольного менеджера в полноценный оконный графический интерфейс (GUI).

Приложение активно развивается, впереди планируются регулярные обновления и добавление новых фич!

## 🚀 Основные возможности
* **Учет баланса:** Автоматический пересчет и отображение общего баланса и итоговой суммы потраченных средств в режиме реального времени.
* **Категории расходов:** Удобный выпадающий список (ComboBox) с 10 предустановленными категориями трат для быстрой классификации.
* **История транзакций (Плитки):** Отображение расходов в виде современных карточек-плиток (режим Tile). На каждой карточке сразу выводятся название траты, сумма и её категория.
* **Сохранение данных в JSON:** Автоматическая сериализация данных при закрытии программы и их десериализация при новом запуске. История и баланс никогда не потеряются.
* **Умные подсказки (Placeholder):** Интеграция нативных исчезающих подсказок в текстовые поля через вызовы функций Windows API (`SendMessage`).
* **Валидация и защита:** Защита от «дурака» — приложение не позволит добавить нулевую трату или пропустить выбор категории.

## 🛠 Технологии и инструменты
* **Язык программирования:** C#
* **Платформа:** Windows Forms (.NET Framework)
* **Формат данных:** JSON (библиотека `System.Web.Script.Serialization`)
* **API операционной системы:** Windows User32 SDK (`user32.dll`)
